## NAMING PHILOSOPHY

**For Monsters:**
- Combine musical term + animal/element
- Examples: Rhythmander, Crescendactyl, Beatle, Harmonize

**For Items:**
- Musical term + weapon/armor type
- Examples: Crescendo Blade, Staccato Shield, Fortissimo Gauntlets

**For Abilities:**
- Musical term + action verb
- Examples: Tempo Theft, Harmony Break, Crescendo Slash

**For Locations:**
- Musical term + geographical feature
- Examples: Crescendo Peaks, Legato Lagoon, Melodic Forest

**For Characters:**
- Use musical terms as names directly
- Personality should match name meaning
- Examples: Allegra (cheerful), Piano (quiet), Forte (strong)

---

## CONSISTENCY RULES

1. **Alliteration is powerful** - Crescendo Chronicles, Fortissimo Fortress
2. **Mix languages** - Italian musical terms + English geography
3. **Evoke feeling** - Harmony = good, Discord = bad
4. **Function suggests name** - Tempo Theft steals speed
5. **Evolution chains share roots** - Tadnote → Froggio → Ribbitar
6. **Cultural diversity** - Not all names need Italian roots
7. **Avoid confusion** - Don't use too-similar names (Harmony/Harmonize)
8. **Pronounceable** - Players should be able to say names
9. **Memorable** - Catchy names stick in mind
10. **Thematic** - Everything sounds like it belongs in same world
