## Usage Guidelines

### Palette Selection:
1. **Character Palettes**: 12-16 colors maximum per character
2. **Environment Palettes**: 16-24 colors per biome/scene
3. **Effect Palettes**: 8-12 colors for VFX

### Color Limitations by Platform:
- **NES-Style**: 8 colors maximum
- **SNES-Style**: 16 colors recommended
- **Modern Hi-Bit**: 24-32 colors

### Mixing Palettes:
- Base environment: Use biome palette (60% of pixels)
- Characters: Use character palette (30% of pixels)
- Effects/UI: Use genre or special palette (10% of pixels)

### Testing Colors:
1. Export palette to Aseprite
2. Test on gray background (#808080)
3. Test contrast with colorblind simulation
4. Verify on actual game backgrounds

---

## Accessibility Notes

**Colorblind-Safe Pairs:**
- Avoid: Red/Green alone
- Safe: Red/Blue, Yellow/Blue, Purple/Orange
- Always use brightness contrast in addition to hue

**Contrast Ratios:**
- UI Text on BG: Minimum 4.5:1 ratio
- Important elements: Minimum 3:1 ratio

All palettes in this library tested for basic accessibility compliance.
