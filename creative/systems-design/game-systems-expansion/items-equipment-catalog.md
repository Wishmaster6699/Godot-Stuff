## Item Set Bonuses

### Crescendo Set (Weapon + Armor + Accessory)
- **2-piece:** Damage increases by 2% per consecutive hit (max 40%)
- **3-piece:** At 20+ combo, enter Crescendo Mode (+50% all stats for 10s)

### Staccato Set
- **2-piece:** Attack speed +30%, damage -10%
- **3-piece:** Every 5th hit is automatic critical

### Fortissimo Set
- **2-piece:** Heavy attacks deal +50% damage
- **3-piece:** Stamina costs reduced by 40% for heavy attacks

### Harmony Set
- **2-piece:** Party members gain +15% all stats
- **3-piece:** Healing effects increased by 50%

### Rhythm Master Set
- **2-piece:** Perfect timing window +30%
- **3-piece:** Perfect hits restore 1% HP and stamina

## System Interactions

**Items → Combat:**
- Buffs enhance combat performance
- Equipment enables different playstyles
- Consumables provide tactical options

**Items → Crafting:**
- Materials fuel crafting economy
- Salvaging creates material loop
- Quality tiers reward mastery

**Items → Progression:**
- Equipment unlocks at certain levels
- Rare items require exploration
- Sets encourage build diversity

**Items → Economy:**
- Items can be sold for profit
- Market prices fluctuate
- Rare materials have high value

---

## Design Philosophy Summary

✅ **Every item should enable a strategy**
✅ **Set bonuses encourage thematic builds**
✅ **Materials create gathering/crafting loop**
✅ **Consumables provide tactical depth**
✅ **Rarity tiers create progression goals**
