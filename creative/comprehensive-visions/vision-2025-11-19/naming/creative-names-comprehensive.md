## Naming Conventions

### Naming Pattern Rules

**Weapon Names:**
- Instrument type + power descriptor
- Example: "Violin Vortex", "Drum of Destiny"

**Ability Names:**
- Musical term + effect
- Example: "Crescendo Strike", "Harmony Burst"

**Location Names:**
- Region identifier + description
- Example: "Rhythmic Forest", "Harmonic Caverns"

**NPC Names:**
- Title/role + Name/descriptor
- Example: "Maestro Veil", "Echo the Songkeeper"

---

## Name Allocation Strategy

### What Gets Named:
1. All major locations (regions, cities, dungeons)
2. All unique NPCs
3. All legendary/unique weapons
4. All major abilities
5. All biomes and regions
6. Plot-critical items
7. Factions and organizations

### What Doesn't Require Full Naming:
1. Generic enemy types (can use descriptor + number)
2. Common consumables (potion generics ok)
3. Random loot (can be procedurally named)
4. Minor NPCs (can use role descriptors)
