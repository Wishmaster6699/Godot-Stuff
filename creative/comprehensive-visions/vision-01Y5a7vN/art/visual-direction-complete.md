# Complete Visual Direction Guide

---

## Art Philosophy

**Core Pillars:**
1. **Musical Expression** - Every visual choice reinforces music/rhythm theme
2. **Readable Rhythm** - Combat clarity is paramount
3. **Genre Diversity** - Different music genres have distinct visual identities
4. **Modern Retro** - Pixel art with contemporary techniques
5. **Emotional Resonance** - Visuals support narrative beats

## Aseprite Workflow

**Canvas Sizes:**
- Tiles: 16x16
- Small items: 16x16
- Player character: 32x32 (or 32x48 for taller)
- Large enemies: 48x48 to 128x128
- Bosses: Up to 256x256
- Backgrounds: Variable (often 320x180 base)

**Layer Organization:**
1. Sketch layer (deleted later)
2. Line art
3. Base colors
4. Shadows
5. Highlights
6. Details
7. Effects/glow
8. Outline (optional)

**Animation Process:**
1. Create key frames
2. Add in-betweens
3. Use onion skinning
4. Test at 60 FPS
5. Add squash/stretch
6. Polish details
7. Export sprite sheet

**Export Settings for Godot:**
- Sprite sheet format: PNG
- JSON/XML metadata
- Trim empty space
- Power-of-2 dimensions (for atlases)
- Separate layers if needed

**Animation Frame Counts:**
- Idle: 4-6 frames (subtle breathing)
- Walk: 6-8 frames
- Run: 8 frames
- Attack: 6-12 frames (depends on weapon)
- Jump: 4 frames
- Land: 3 frames
- Hurt: 2 frames
- Death: 6-8 frames
- Special moves: 12-16 frames

**Production Pipeline:**
1. Concept sketch (rough)
2. Clean line art
3. Block colors
4. Shading pass
5. Detail pass
6. Animation frames
7. VFX addition
8. Export and integrate
9. Test in-game
10. Polish iteration

## Unique Selling Points

1. **Genre Visual Identity** - Each music genre has distinct art style
2. **Rhythm-Reactive Visuals** - Everything pulses to beat
3. **Musical UI** - Interfaces that look like instruments
4. **Expressive Animation** - Characters show rhythm state
5. **Modern Pixel Art** - Contemporary techniques (lighting, particles)
6. **Cultural Respect** - Authentic representation of musical traditions
7. **Accessibility** - Visual options for color-blind, photosensitivity
8. **Customization** - Player visual identity matters

**Innovation:** First rhythm game where visual design teaches musical concepts (genre, theory, history) through aesthetics alone.
